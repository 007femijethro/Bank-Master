1) Admin balance control (ledger-safe)
Instead of “editing balance directly”, make admin do adjustment transactions (this is how real systems protect integrity):
Admin can add, remove, or set balance via:
Adjustment Credit (adds)
Adjustment Debit (removes)
Balance Correction (set balance by posting a correction entry)
Every action requires:
reason code + notes
staff user id
audit log entry
✅ This keeps your app defendable: “balances are derived from ledger entries.”
